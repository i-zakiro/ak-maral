Стандартный двухместный номер представляет с собой однокомнатный номер с двумя раздельными односпальными кроватями, тумбой под чемодан и вместительным шкафом, так же номер оснащен телевизором, холодильником для охлаждения напитков (по желанию гостей оснащаемым мини баром), сейфом для хранения документов, ценностей и денежных средств, с сан узлом (ванна, туалет).
В номере:
Cпутниковое телевидение
Банный халат
Тапочки
Средства личной гигиены
Набор полотенец из четырех штук (полотенце пляжное, полотенце банное, полотенце для рук, полотенце для ног)
Электрочайник с посудой для комфортного чаепития
